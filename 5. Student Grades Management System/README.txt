Student Grades Management System

📌 Description

A simple C++ console application to manage student grades, including:

Adding grades

Calculating average

Finding highest & lowest grades

File storage & retrieval

🚀 Usage

Compile & Run:

g++ main.cpp -o student_grades
./student_grades

📂 File Structure

📦 StudentGrades
 ┣ 📂 Data (Stores student grade data)
 ┣ 📜 main.cpp (Code)
 ┣ 📜 README.md (Docs)
 ┗ 📜 .gitignore (Ignore unnecessary files)
