A simple number guessing game written in C++.

## Features
- Generates a random number between 1 and 1000.
- Allows the user to guess the number with hints.
- Keeps track of the number of attempts.
- Option to exit the game at any time.

## How to Play
1. Run the program.
2. Choose **1** to start the game.
3. Enter a number between 1 and 1000.
4. The game will guide you with:
   - `"Too high."` if your guess is too high.
   - `"Too low."` if your guess is too low.
   - `"You won!"` when you guess correctly.
5. Enter **-1** anytime to exit the game.
